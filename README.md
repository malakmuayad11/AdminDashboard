# 🧑‍💻 AdminDashboard

A responsive Admin Dashboard UI built with HTML, TypeScript, and Tailwind CSS.
It includes a collapsible sidebar, analytics cards, charts, and a user management table.

## ✨ Features

- 📱 Fully responsive layout (mobile + desktop)
- 📂 Collapsible sidebar with smooth animation
- 🔎 Search bar in top navigation
- 📊 Dashboard statistics cards (Users, Revenue, Orders, Growth)
- 📈 Revenue line chart (SVG-based)
- 📊 User activity bar chart
- 👥 Recent users table with actions (view/edit UI)
- 🎨 Styled using Tailwind CSS utility classes
- ⚡ TypeScript-based UI interactions

## 🛠️ Technologies

- HTML5
- Tailwind CSS v4
- TypeScript
- JavaScript (ES Modules)
- SVG (for charts)
- Google Fonts (Inter)

## 🚀 How to Run Locally

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/AdminDashboard.git
   cd AdminDashboard
   ```
2. Install dependencies:
   ```bash
    npm install
    ```
4. Build TypeScript:
   ```bash
    npx tsc
   ```
6. Build Tailwind CSS (If you're using Tailwind CLI):
   ```bash
     npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
   ```
8. Open the project: open index.html in your browser.

### ⚙️ Notes
- dist/ is auto-generated (do not edit manually)
- TypeScript compiles from src/script.ts → dist/script.js
- Tailwind compiles from src/input.css → dist/output.css
